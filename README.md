# 🍉 Fruit Ninja Game
A fun and interactive Fruit Ninja-inspired game built with **HTML**, **CSS**, and **JavaScript**.

## 🎮 Features
- 🍎 Slice fruits with the mouse
- 💣 Avoid bombs
- ❤️ 3 Lives system
- 🏆 High Score using Local Storage
- 🎵 Sound effects
- 🔥 Combo score bonus
- 📱 Responsive design
- 🎨 Attractive animations
- 🚀 Fast and lightweight

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas

## ▶️ How to Run
1. Clone the repository
2. Open the project folder.
3. Double-click **index.html** or run it using **Live Server** in VS Code.

## 🎯 Game Rules
- Slice fruits to earn points.
- Don't hit bombs.
- Missing fruits reduces lives.
- Game ends when all lives are lost.
- Beat your high score!
